Welcome to the Vector documentation. Whether you're just getting started or looking to unlock advanced workflows, you'll find everything you need here.

## What is Vector?

Vector is an AI coding agent that lives in your editor and your terminal. It can read and write files, run terminal commands, use a browser, and help you build features through natural conversation. Every action requires your explicit approval. You're always in control.
